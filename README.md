## What Test Commander Is

Test Commander is a Claude Code skill pack and methodology for moving from product understanding to exploratory testing, test design, BDD documentation, Playwright automation, and quality reporting.

It helps Claude Code behave less like a random test generator and more like a disciplined testing assistant.

## What Test Commander Is Not

- It is not a replacement for skilled testers.
- It is not a fully autonomous QA system.
- It is not a promise that AI can understand every product perfectly.
- It is not a test automation silver bullet.

It is a structured workflow for making AI-assisted testing safer, more useful, and more traceable.
